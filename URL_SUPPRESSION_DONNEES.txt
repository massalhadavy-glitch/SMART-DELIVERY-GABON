URL pour Google Play Console - Demande de Suppression de Données

===========================================
OPTION 1 : Page HTML (à héberger)
===========================================

Une page HTML a été créée : demande_suppression_donnees.html

Pour l'utiliser :
1. Hébergez ce fichier sur un serveur web (GitHub Pages, Netlify, Vercel, etc.)
2. Utilisez l'URL complète dans Google Play Console

Exemple d'URL après hébergement :
https://votre-domaine.com/demande_suppression_donnees.html

===========================================
OPTION 2 : Google Forms (Rapide et Gratuit)
===========================================

1. Créez un formulaire Google Forms
2. Ajoutez les champs nécessaires :
   - Identifiant/Email
   - Message de confirmation
3. Partagez le formulaire et copiez l'URL

URL à utiliser dans Google Play :
https://forms.gle/VOTRE_ID_FORMULAIRE
